# Security Policy

## Reporting

Please do not open public issues for sensitive reports.

For security concerns, contact the repository owner privately with:

- A concise description of the issue.
- Affected version or release artifact.
- Reproduction context when appropriate.
- Any relevant logs with personal information removed.

## Scope

This public repository is release-oriented and does not include the complete private implementation or build environment.

## Public Artifacts

Only download APK files from the official GitHub Releases page for this repository. Verify release checksums when provided.
